🦠 Flu Syndrome Prediction — pred_sindromes_gripais
A machine learning project to predict flu syndrome occurrences using a Support Vector Regression (SVR) model, built with Python and Jupyter Notebook.

📌 Overview
Flu syndromes (síndromes gripais) are a recurring public health concern in Brazil, with seasonal spikes that strain health systems. This project explores whether historical epidemiological data can be used to predict future case counts using supervised machine learning.

🧠 Model

Algorithm: Support Vector Regression (SVR)
Library: scikit-learn
Environment: Jupyter Notebook
